## 1. **GitHub Pages + Statische Generatoren** (kostenlos, beliebt)
- **Jekyll / Hugo / Astro / VitePress** – bauen statische Seiten aus Markdown/Daten, hosten kostenlos auf GitHub Pages.
- Datenquelle: GitHub REST/GraphQL API (per GitHub Action regelmäßig gefetched → JSON/CSV → Charts).
- Charts: **Chart.js**, **ApexCharts**, **Recharts**, **Observable Plot**, **Vega-Lite**.

## 2. **Observable / Observable Framework**
- Notebook-ähnliche, reactive Dashboards.
- Kann direkt GitHub API abfragen (mit Token) oder vorberechnete Daten laden.
- Hosting: **Observable Cloud** (kostenloser Tier) oder **Observable Framework** → statischer Export → GitHub Pages.

## 3. **Grafana** (wenn du einen Server hast)
- **GitHub Datasource Plugin** (Community) oder **Infinity / JSON API** Plugin.
- Läuft auf eigenem Server / Docker / Grafana Cloud (Free Tier: 3 Nutzer, 50 GB Logs/Metrics).

## 4. **Apache Superset / Metabase / Evidence.dev**
- Open-Source BI-Tools, die SQL/CSV/JSON visualisieren.
- Evidence.dev ist code-first (Markdown + SQL), baut statische Seiten → GitHub Pages.

## 5. **Streamlit / Panel / Dash / Shiny** (Python/R)
- Interaktive Apps, Deployment auf **Streamlit Community Cloud**, **Hugging Face Spaces**, **Render**, **Fly.io** (alles kostenlose Tiers).
- Daten via `PyGithub` / `githubkit` / GraphQL holen.

## 6. **GitHub Insights / Built-in** (nur Basics)
- Pulse, Contributors, Traffic, Dependency Graph – reicht oft für einfache Übersichten.

## 7. **Custom: GitHub Actions → Artefakt → GitHub Pages**
- Workflow läuft täglich/stündlich, ruft API ab, schreibt `data.json`, Commit & Push → Pages baut Dashboard (z. B. mit Vite + Vue/React/Svelte + Charts).
